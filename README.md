# Snake Game

## Overview
The Snake Game is a classic arcade game where the player controls a snake to eat food, which makes it grow longer. The goal is to avoid colliding with the walls or the snake's own body while trying to eat as much food as possible.

## Features
- **Classic Gameplay**: Traditional snake movement and growing mechanics.
- **Score Tracking**: Keep track of the highest score during the game session.
- **Responsive Controls**: Move the snake using keyboard arrow keys.
- **Game Over Conditions**: Detect collisions with walls and the snake's body.

## Getting Started
To run the Snake Game locally:
1. Clone this repository:
   ```bash
   git clone https://github.com/Wajeed-Mabroukeh/Snake-Game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Snake-Game
   ```
3. Open the `index.html` file in your web browser.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Start the game by opening the `index.html` file in your browser.
2. Use the arrow keys to move the snake:
   - Up Arrow: Move Up
   - Down Arrow: Move Down
   - Left Arrow: Move Left
   - Right Arrow: Move Right
3. Eat the food to grow the snake.
4. Avoid colliding with the walls and your own tail.
